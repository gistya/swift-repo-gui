# Tracker Module Sources

These tracker modules are bundled as source/provenance material for the SwiftBuilder soundtrack.
They are listed by The Mod Archive under its Public Domain license browser.

AVFoundation does not natively decode XM/IT/MOD tracker modules, so the app currently uses an
in-app tracker-style renderer for playback. These original modules are kept here for a future
libopenmpt-backed player.
